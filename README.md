# JWT Golang

A go (or 'golang' for search engine friendliness) implementation of JSON Web Tokens.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/brf153/jwt-golang
   ```
2. Install Go packages
   ```sh
   go mod vendor
   ```
